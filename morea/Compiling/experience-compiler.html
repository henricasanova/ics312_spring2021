Homework Assignment #9 [40 pts]

You are expected to do your own work on all homework assignments. You may (and are encouraged to) engage in general discussions with your classmates regarding the assignments, but specific details of a solution, including the solution itself, must always be your own work. (See the statement of Academic Dishonesty on the Syllabus.)

How to turn in?

Assignments need to be turned in via Laulima. Check the Syllabus for the late assignment policy for the course.

What to turn in?

You must turn in one .g4 file called MyLanguageV1Code.g4 that implements your compiler.

Download starter .g4 file

Download the MyLanguageV0Code.g4 file available here, and rename it as MyLanguageV1Code.g4. Replace V0 by V1 inside that file (line 3).

Each question in the assignment asks you to implement a feature of the language but you turn in a single file that implements all the features you were able to implement, in sequence.
