Learning the Linux command line

In this course, some knowledge of the Linux command line is required. Being proficient with the command line is a very valuable skill (for future courses, and in most professional contexts).

To help those of you with little or no experience here are some useful pointers:

  • A set of Shell tutorial exercises in the browser
  • Learning the command-line at the Code Academy (you will have to sign up for a free account)
  • The low-tech Terminus Game to learn how to navigate the file system
  • A set of lecture notes I put together at some point (unsorted and random!)
  • A tutorial with tons of useful links at http://linuxcommand.org
  • A comprehensive tutorial

Also, do not hesitate to ask question about this!


When I use the command line in class and you see me do something that you don't get or don't know about, ASK! :)
